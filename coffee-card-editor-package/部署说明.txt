# 咖啡名片编辑器 - 部署说明

## 📦 打包内容
- index.html (主页面)
- style.css (样式文件)
- script.js (主要逻辑)
- server.js (Node.js服务器)
- package.json (项目配置)
- img/ (图片文件夹)
- README.md (说明文档)

## 🚀 宝塔面板部署步骤

### 1. 上传文件
1. 登录宝塔面板
2. 进入"文件"管理
3. 上传整个 coffee-card-editor-package 文件夹到网站根目录
4. 解压或直接使用

### 2. 安装Node.js环境
1. 在宝塔面板中安装Node.js (建议版本 16+)
2. 进入"软件商店" → "运行环境" → 安装Node.js

### 3. 安装依赖
1. 在宝塔面板终端中进入项目目录
2. 运行命令: npm install

### 4. 启动服务
1. 在宝塔面板中创建Node.js项目
2. 项目路径: 选择您的项目目录
3. 启动文件: server.js
4. 端口: 3000 (或您想要的端口)
5. 点击"启动"

### 5. 配置域名 (可选)
1. 在宝塔面板中添加域名
2. 配置反向代理到 localhost:3000
3. 或直接使用 IP:端口 访问

## 🌐 访问地址
- 本地访问: http://localhost:3000
- 服务器访问: http://您的服务器IP:3000
- 域名访问: http://您的域名 (如果配置了域名)

## 📝 注意事项
1. 确保服务器防火墙开放3000端口
2. 图片文件放在 img/ 文件夹中
3. 日志文件会自动保存在 log/ 文件夹中
4. 建议使用PM2管理Node.js进程

## 🔧 常用命令
- 启动服务: node server.js
- 安装依赖: npm install
- 查看日志: 在宝塔面板中查看Node.js项目日志
